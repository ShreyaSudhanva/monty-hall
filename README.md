# Monty Hall Live Lab

An interactive Monty Hall experiment built with React + Vite. Play a round, watch the host reveal a goat, and see the math show up in the strategy dashboard. Includes a multi-run simulation to validate the switch advantage in seconds.

## Features
- Animated host reveal with flipping doors
- Strategy stats dashboard (stay vs switch)
- Multi-run simulation with configurable strategies
- Responsive layout with bold visual design

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
